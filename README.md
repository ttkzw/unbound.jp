# unbound.jp web site

これはウェブサイト https://unbound.jp/ のリポジトリです。

静的サイトジェネレーター[Astro](https://astro.build/)と[Starlight](https://starlight.astro.build/)を利用してページを生成します。

## デプロイ

```
git clone https://github.com/ttkzw/unbound.jp.git
npm install
```

## 静的ファイル生成

```
npm run build
```

## 開発用サーバーの起動

```
npm run dev
```

## アップデート

```
npx @astrojs/upgrade
```
